# ewf

A greenhouse monitor and controller for the Arduino MEGA 2560.

### Features

- real time clock
- time based switching
- delay based switching
- PWM dimming
- temperature monitoring (SHT31, RTC/DS3231, DS18B20 probe)
- humidity monitoring
